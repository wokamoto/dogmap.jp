== Installation ==

1. Upload the entire `wp-cron-dashboard` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

The control panel of Head Cleaner is in 'Tool > WP-Cron'.
